# Git Log

Il comando *git log* non fa niente di particolare, ma ci permette di vedere i log delle commit.

commit a5cd2e88877c6016e36d2046633d067fcd78fcbd
Author: Christian Luzzetti <christian.luzzetti@gmail.com>
Date:   Wed May 5 22:48:59 2021 +0200

    Rinominato il file


per ogni commit, posso vedere svariate impostazioni, tra cui il messaggio di commit, il nome dell'autore della commit, la data, ed un hash.
L'hash è importante, ma vedremo più in la il motivo.
